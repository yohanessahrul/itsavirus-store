## Run Locally

Clone the project

```bash
  git clone https://github.com/yohanessahrul/itsavirus-store.git
```

Go to the project directory

```bash
  cd itsavirus-store
```

Install dependencies

```bash
  npm install sass
```

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

  
## Tech Stack

**Client:** NextJs, React, Redux, Sass
